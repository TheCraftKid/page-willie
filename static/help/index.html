Frequently Asked Questions

Why does Page Willie use JavaScript?

TL;DR: Page Willie uses JavaScript to handle requests and fancy site functionality.

Page Willie uses javaScript for the following features:

  • Sending requests
  • Getting notifications on requests
  • Fetching list of app updates
  • Analytics to improve the site

What's a cookie and why do you use them?

TL;DR: Cookies are small pieces of information stored for website by your browser.

Page Willie uses cookies for Google Analytics, an analytics software that tells me who's using the site and how you're using it. I use this data to improve site functionality like placement of items on screen or bug fixes. Cookies are also used by advertisers to allow more personalized ads.

Of course, for the privacy-minded, cookies can be turned off, but you'll have to sign in every time you open the site.
